# Research Skills Assistant

An interactive web application designed to help students and researchers develop essential research skills, including formulating research titles, writing problem statements, and crafting research significance.

## Features

- 📚 Research Title Builder
- 🎯 Problem Statement Generator
- 💡 Research Significance Composer
- 🔍 Literature Search Integration with OpenAlex API
- 📊 Interactive UI with real-time validation
- 🎨 Professional modern design

## Deployment

This project is hosted on Vercel and deployed automatically with each push to the main branch.

**Live Demo:** [Visit the app](https://interactive-research-skills-assistant-n41r9tyoc.vercel.app)

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. Start developing!

No build server needed - this is a static HTML/CSS/JavaScript project.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Lucide Icons
- OpenAlex API
